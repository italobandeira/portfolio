# Ítalo Bandeira | Portfolio 🚀

> **Engenheiro de Software Full-Stack | Designer | Motion | IA**

Bem-vindo ao repositório do meu portfólio pessoal. Este projeto é uma vitrine visual e interativa do meu trabalho, desenvolvido sob os conceitos estéticos do **Brutalismo Desconstruído**. Ele combina design ousado, layouts fluidos e animações dinâmicas para proporcionar uma experiência de usuário única e impactante.

🔗 **Visualizar Projeto:** [Portfólio Online](https://italobandeira.github.io/portfolio/) *(se estiver hospedado no GitHub Pages)*

---

## 🎨 O Projeto: Estética e Conceito

A interface foge dos layouts tradicionais corporativos, adotando um estilo brutalista moderno. O uso de grids aparentes, fitas adesivas ("tapes") ilustrativas, grandes tipografias, efeitos *grayscale* com ativação via *hover* e seções no estilo terminal (*CLI*) comunicam a mescla entre **design visual avançado** e **engenharia de software robusta**.

As seções incluem:
- **Hero & Manifesto:** Apresentação de impacto destacando minhas principais áreas de atuação.
- **Skills (Marquee):** Letreiros contínuos exibindo meu *tech stack*.
- **Projetos Full-Stack:** Cards de projetos em produção com links diretos de acesso.
- **Galeria Design & Motion:** Carrosel horizontal infinito (Marquee) apresentando portfólio criativo com visualizador integrado (Lightbox).
- **Terminal & Contato:** Interface visual via comandos de terminal com links para redes e email.

---

## ⚙️ Stack Tecnológico

O projeto foi construído pensando em velocidade de carregamento, responsividade e facilidade de manutenção de uma aplicação puramente Client-Side, utilizando:

- **HTML5:** Estrutura semântica;
- **Tailwind CSS (via CDN):** Estilização rápida, eficiente e responsiva (Mobile First);
- **Vanilla JavaScript:** Lógicas de animação, manipulação do DOM e Lightbox nativo;
- **Google Fonts:** Tipografia utilizando `Inter`, `Space Grotesk` e `Space Mono`;
- **Google Material Symbols:** Ícones vetoriais embutidos;
- **OWASP Best Practices:** Atributos seguros em links externos (`rel="noopener noreferrer"`) e meta tags focadas em performance/segurança.

## 🛠 Como executar localmente

Como o projeto utiliza tecnologias puramente web e estáticas, rodar localmente é muito simples:

1. Clone o repositório:
```bash
git clone https://github.com/italobandeira/portfolio.git
```
2. Navegue até a pasta do projeto:
```bash
cd portfolio
```
3. Abra o arquivo `index.html` em seu navegador.
   - *Alternativa:* Utilize a extensão **Live Server** do VSCode para emular um servidor local instanciando as requisições *client-side*.
