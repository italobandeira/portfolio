## 🎨 O Projeto: Estética e Foco

A interface foi projetada para resolver um problema comum: sites lentos e genéricos que não geram leads. A landing page utiliza a cor `Accent` (Amarelo Neon) para guiar o olhar do usuário diretamente para as Call-to-Actions (CTAs) e destacar os diferenciais.

As seções incluem:
- **Hero:** Apresentação de alto impacto com efeito de falha (Glitch), background de partículas dinâmicas interagindo com o cursor e Copy orientada ao fechamento de negócios.
- **Benefícios (Marquee):** Letreiro contínuo de alta fluidez destacando pilares como "Alta Conversão", "SEO Otimizado" e "Carregamento Rápido".
- **Projetos Trabalhos Recentes:** Grid de projetos com proporção balanceada e mudança dramática de "grayscale" para "colorido" no hover para alto engajamento. (Destaques: Advocacia, Energia Enxuta e Betterday).
- **Sobre & Especialidades:** Reforço de autoridade no nicho de conversão, combinando Design e Engenharia de Software rápida.
- **Contato Direto:** Opções diretas de fechamento, com ênfase no e-mail corporativo e WhatsApp.

## ⚙️ Stack Tecnológico

Visando máxima velocidade de carregamento — crucial para SEO e campanhas de Tráfego Pago —, a stack utiliza de ferramentas que eliminam a necessidade de configurações de build na máquina do cliente, rodando perfeitamente "Client-Side":

- **HTML5:** Estrutura simples e semântica;
- **Tailwind CSS (via Play CDN):** Todo o design system, utilitários, responsividade (*Mobile First*) e animações UI criados diretamente no HTML.
- **Vanilla JavaScript:** Lógicas de Custom Cursors e comportamento do viewport.
- **Three.js:** Framework 3D (`WebGLRenderer`, `ShaderMaterial`) utilizado no Background para renderizar partículas complexas responsivas ao movimento do mouse num plano cartesiano fluido.
- **Google Fonts:** Tipografia moderna unindo as fontes de estilo técnico `Space Mono` e corporativo legível `DM Sans`.

## 🛠 Como executar localmente

Sendo um projeto web completamente estático construído com o CDN do Tailwind, não há necessidade do `NPM` ou processos de transpilação.

1. Clone o repositório:
```bash
git clone https://github.com/italobandeira/portfolio.git
```
2. Navegue até a pasta do projeto:
```bash
cd portfolio
```
3. Simplesmente abra o arquivo `index.html` em qualquer navegador.
   - *Dica:* Utilize a extensão **Live Server** no editor de código da sua preferência para ter auto-reload dinâmico enquanto edita o HTML.
